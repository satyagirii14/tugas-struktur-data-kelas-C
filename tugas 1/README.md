# Penjelasan program 
## - Program meminta 10 nilai dari pengguna dan menyimpannya ke dalam list. Karena harus membaca semua input, waktu yang dibutuhkan sebanding dengan jumlah nilai. Jadi, semakin banyak nilai, semakin lama prosesnya.
- Mencari nilai terbesar dan terkecil: Fungsi max() dan min() harus melihat semua isi list untuk menemukan hasilnya. Artinya, waktu yang dipakai juga sebanding dengan jumlah nilai.
- Menghitung rata-rata: Fungsi sum() menjumlahkan semua nilai satu per satu, lalu dibagi dengan jumlah data. Lagi-lagi, waktunya bergantung pada banyaknya nilai.
- Menghitung jumlah lulus: Program memeriksa setiap nilai apakah ≥ 60. Karena harus mengecek semua, waktunya juga sebanding dengan jumlah nilai.
- Menghitung jumlah tidak lulus: Ini hanya operasi pengurangan sederhana, jadi waktunya tetap sama meskipun jumlah nilai bertambah.
